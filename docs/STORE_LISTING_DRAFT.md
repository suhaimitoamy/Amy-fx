# Store Listing Draft

## App Name

Amy FX

## Short Description

Trading analysis suite untuk XAU/USD, mapping ICT, scanner target, jurnal, academy, dan library indikator.

## Full Description

Amy FX membantu trader melakukan analisis XAU/USD, menyimpan jurnal trading, memantau level penting, dan mengelola library indikator lokal.

Fitur utama:

- Mapping market XAU/USD.
- ICT structure, FVG, OB, BSL/SSL helper.
- Background scanner target.
- Notifikasi target.
- Jurnal trading.
- Library indikator Pine Script.
- Candle cache lokal.

Catatan: Amy FX bukan aplikasi eksekusi order otomatis dan bukan nasihat keuangan.
