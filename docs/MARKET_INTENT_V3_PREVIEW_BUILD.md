# Amy FX Preview — Market Intent V3

Build target: `1.4.15-preview.2` (`90039`)

Fitur yang harus terlihat saat Mapping dibuka:

- Pertanyaan utama: **Market mau ke mana?**
- Arah utama: naik ke BSL, turun ke SSL, atau arah belum bersih.
- Primary dan secondary liquidity objective.
- Batas invalidasi atau sisi liquidity lawan.
- Expected Price Path empat tahap.
- Market Regime, Market Shift, Strategy Router, dan keputusan eksekusi M15.
- Mode Fokus dan Detail Teknis.

Build memakai package `com.amyelitesuite.learningpreview` dan signer preview permanen. Branch `main` tidak diubah dan tidak boleh di-merge sebelum evaluasi pengguna selesai.
