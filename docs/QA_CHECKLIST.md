# QA Checklist

```text
[ ] App bisa install fresh di Android 7, 10, 12, 14, dan 15.
[ ] Dashboard utama terbuka tanpa blank screen.
[ ] Mapping bisa mengambil candle.
[ ] Scanner bisa start dan stop.
[ ] Target BSL/SSL tersimpan dari Mapping.
[ ] Notifikasi BSL/SSL muncul saat level tersentuh.
[ ] Tap notifikasi membuka Mapping.
[ ] Cooldown notifikasi berjalan 30 menit per level.
[ ] Target lama otomatis expired setelah 24 jam.
[ ] Jurnal bisa add, edit, dan delete entry.
[ ] Indikator bisa dibuka dan disalin.
[ ] Export file tersimpan di Download.
[ ] App tidak crash saat offline.
[ ] Unit test lulus.
[ ] Release APK tidak berisi credential rahasia.
```
