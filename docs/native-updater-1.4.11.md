# Amy FX 1.4.11 Native Updater

- Download APK langsung di dalam aplikasi.
- Progress persen dan ukuran file.
- Cache internal, bukan folder Download.
- Verifikasi package, version code, dan signer SHA-256.
- FileProvider untuk installer Android.
- Izin instalasi sumber tidak dikenal diminta per aplikasi.
- Browser hanya fallback untuk versi lama yang belum memiliki native bridge.
