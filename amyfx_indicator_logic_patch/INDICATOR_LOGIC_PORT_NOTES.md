# Amy FX Mapping - ICT Indicator Logic Port

Patch ini mengganti logika Mapping agar mengikuti fitur indikator ICT/SMC yang diberikan:

- Market Structure: Internal & Swing BOS / CHoCH / MSS context
- Order Blocks: internal/swing OB, mitigation High/Low atau Close
- Fair Value Gap: bullish/bearish FVG, touched/broken, BPR overlap
- Liquidity: BSL/SSL grouped liquidity zones
- EQH/EQL: equal highs/lows berdasarkan ATR threshold
- Volume Imbalance
- NWOG/NDOG
- Premium/Discount/Equilibrium zones
- Killzones Auto DST
- Multi-timeframe group analysis
- MTF previous levels engine
- Displacement candle filter
- Signal/context engine untuk BUY CONTEXT, SELL CONTEXT, WATCHLIST, WAIT
- Target BSL/SSL dikirim ke native scanner

Dashboard tidak lagi menampilkan panel Analyze penuh. Analyze tetap berada di tab Analyze.
