# Privacy Policy

Amy FX menyimpan data utama secara lokal di perangkat pengguna.

## Data yang Disimpan

- API key market data.
- Target scanner BSL/SSL.
- Candle cache.
- Jurnal trading.
- Preferensi aplikasi.

## Penyimpanan

API key disimpan menggunakan encrypted preferences. Data jurnal dan cache disimpan lokal kecuali pengguna mengaktifkan backup eksternal.

## Data Pihak Ketiga

Aplikasi dapat menghubungi layanan market data seperti TwelveData atau Supabase jika pengguna mengatur API/endpoint.
