Premium Discount Array (PD Array)

Ikhtisar

PD Array atau matriks harga adalah hierarki elemen teknikal ICT (seperti Order Blocks, Fair Value Gaps, Breaker, dll) yang disusun berdasarkan kepentingannya dari titik ekuilibrium (50%) menuju titik ekstrim.

Apa itu PD Array?

Ini adalah "alat bantu" (susunan zona/level) yang digunakan Algoritma untuk menahan atau memantulkan harga. Mereka dikategorikan menjadi dua: - Premium Arrays: Terdapat di atas level 50%, digunakan untuk mencari posisi Sell. - Discount Arrays: Terdapat di bawah level 50%, digunakan untuk mencari posisi Buy.

Hierarki PD Array (Dari Luar ke Dalam)

  1. Old High/Old Low (Liquidity)
  2. Rejection Block
  3. Order Block
  4. Fair Value Gap (FVG)
  5. Breaker Block
  6. Mitigation Block

Mengapa Menggunakan PD Array?

Ini menyaring di mana institusi kemungkinan besar memiliki minat (interest). Jika kamu ingin membeli, kamu tidak boleh membeli di sembarang support. Kamu harus menemukan Discount Array yang valid.

Bagaimana Menggunakannya?

  1. Tentukan Dealing Range.
  2. Bagi dua menjadi Premium dan Discount.
  3. Jika bias kamu adalah Buy, tandai semua Discount Array. Jika harga turun, lihat array mana yang pertama kali bereaksi memberikan Market Structure Shift di LTF.

Kriteria Trading

  • Entry: Di salah satu PD Array yang searah dengan Institutional Order Flow HTF.
  • Invalidasi (SL): Di bawah ujung Dealing Range ekstrim.
  • Target (DOL): Premium/Discount Array di sisi yang berseberangan.
Ikhtisar
Pesan Mentor Tidak semua PD Array akan memantulkan harga. Algoritma akan memilih satu yang sejalan dengan "Naratif Waktu". Kombinasikan PD Array ini dengan Killzones (Waktu sesi yang tepat) untuk mendapatkan akurasi mematikan!
