Cara Setting TradingView untuk SMC

Ikhtisar

Agar analisa Smart Money Concept (SMC) dan ICT Anda presisi, workspace TradingView Anda harus bersih dan objektif.

1. Bersihkan Chart (Naked Chart)

Trader institusi tidak menggunakan indikator osilator seperti RSI, MACD, atau Stochastic. - Hapus semua indikator lagging. - Sisakan hanya pergerakan harga (price action) berupa Candlestick.

2. Pengaturan Warna Candlestick

Gunakan warna yang tidak memicu emosi (seperti merah terang atau hijau menyala). Warna netral membantu psikologi Anda tetap stabil. - Bullish Candle: Biru pastel, putih, atau abu-abu terang. - Bearish Candle: Hitam, abu-abu gelap, atau biru tua. - Background: Abu-abu solid atau hitam (Dark Mode) tanpa grid lines (garis-garis latar belakang). Matikan grid lines di Settings > Appearance > Grid lines (opacity 0%).

3. Simpan Template Drawing Tools

Buat favorite toolbar untuk alat gambar yang sering digunakan: 1. Trendline: Gunakan untuk menandai Liquidity (BSL/SSL). 2. Horizontal Ray: Untuk menandai level penting seperti PDH/PDL (Previous Day High/Low). 3. Rectangle: Untuk menggambar Order Block (OB) dan Fair Value Gap (FVG). - Tips: Simpan template warna berbeda (misal: FVG Biru untuk Bullish, FVG Merah untuk Bearish). 4. Fibonacci Retracement: Setting hanya level 0, 0.5 (Equilibrium), 1, dan level OTE (0.62, 0.705, 0.79). 5. Text Tool: Untuk memberikan anotasi CHoCH, MSS, BOS, atau IDM.

4. Pengaturan Zona Waktu (Timezone)

Sangat krusial dalam konsep ICT: - Ubah zona waktu TradingView Anda di pojok kanan bawah ke UTC-4 (New York). - Ini wajib agar Anda bisa selaras dengan waktu pembukaan dan penutupan Session New York, serta menentukan True Day Open dan Midnight Open.
